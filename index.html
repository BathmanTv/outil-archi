Assistant d'agencement

Suggestions par règles (contraintes + programme). Déterministe, hors-ligne, sans IA.

Comment utiliser l'assistant

  1. Crée ton plan d'abord dans l'onglet « Plan 2D » : ajoute les pièces et pose les contraintes techniques (eau, élec, sortie secours, mur porteur…).
  2. Choisis le projet à analyser dans la liste à gauche.
  3. Coche « ERP / commercial » si c'est un commerce — active les conseils sortie de secours, circulations PMR et flux client.
  4. Décris le programme (optionnel) : ex. « boutique 60 m² : vitrine, caisse, 2 cabines, réserve ». Plus c'est précis, plus c'est ciblé (un « restaurant » déclenche les conseils extraction/CHR…).
  5. Plan de référence (optionnel) : uploade une image de plan pour l'avoir sous les yeux.
  6. Clique « Proposer un agencement ».

Résultat : un résumé du lieu, des recommandations par thème (réseaux, lumière, circulation, flux, structure…) et les points de vigilance. Suggestions par règles — fiable, hors-ligne, sans IA.
